## Project Overview
This project aims to develop a machine learning model to accurately predict the 'Dependent_Variable' based on the provided dataset. The project involves various steps, including data preprocessing, exploratory data analysis (EDA), feature engineering, model selection, and evaluation.

## Steps followed 
1. Data Loading
2. Handling Missing Values
3. Exploratory Data Analysis
4. Feature Selection/Engineering
5. Outlier Detection and Handling
7. Variance Thresholding
8. Standardization
9. Model Selection
10. Model Evaluation
11. Prediction on Test Data
